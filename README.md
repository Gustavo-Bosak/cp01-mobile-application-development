# 🎓 CP1 - Cadastro de Alunos

Aplicação mobile desenvolvida com **React Native** e **Expo** para cadastro de informações estudantis. O projeto apresenta uma interface moderna, inspirada no *Dark Mode* do GitHub, utilizando componentes nativos e gerenciamento de estado. O projeto é destinado para primeira avaliação (CP1) da disciplina de Mobile Application Development.

---

## 🚀 Funcionalidades

- **Formulário de Cadastro:** Campos para Nome, Curso, Disciplina e uma descrição multi-linha.
- **Feedback por Modal:** Exibição dos dados preenchidos em uma janela sobreposta (Modal) após o envio.
- **Limpeza de Dados:** Função para resetar os campos do formulário ao fechar o modal.
- **Interface Responsiva:** Uso de `SafeAreaView` para garantir a visibilidade em diferentes dispositivos.

## 🛠️ Tecnologias e Dependências

- **Framework:** [Expo](https://expo.dev/) (v54.0.33)
- **Biblioteca Base:** [React Native](https://reactnative.dev/) (v0.81.5)
- **React:** v19.1.0

## 📦 Como Instalar e Rodar

1. **Clone este repositório:**
   ```bash
   git clone [https://github.com/Gustavo-Bosak/cp01-mobile-application-development.git](https://github.com/Gustavo-Bosak/cp01-mobile-application-development.git)
    ```
2. Acesse a pasta do projeto:
    ```bash
    cd seu-repositorio
    ```

3. Instale as dependências:
    ```bash
    npm install
    ```

4. Inicie o servidor de desenvolvimento:
    ```bash
    npx expo start
    ```

5. Visualize no dispositivo:

- Abra o app Expo Go no seu celular e escaneie o QR Code.

- Ou pressione a para Android, i para iOS ou w para Web.
